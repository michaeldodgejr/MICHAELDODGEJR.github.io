City & Resort Hotel Revenue Analysis

[SQL | Power BI]

For this project I utilized Microsoft SQL Server to manipulate the data and prepare it for use in Power BI. During this process, I used multiple JOIN operations to combine my database tables. I connected my database server to Power BI accompanied by SQL statements which made the data ready for visualization. I then developed an interactive dashboard used to answer the objectives of this analysis:

• Identify if the hotel revenue is growing by year.
• Determine if the parking lot spaces should be increased.
• Discover trends in the data utilizing ADR and seasonality.

Some key findings in this analysis are that there is a clear upward trend in revenue from 2018 to 2020. Looking further into the data you can see that Resort hotels utilize more parking lot spaces, but they do not need to be increased for either hotel type. The sharp increase in the average daily rate from 2019 to 2020 suggests that the hotel industry has been able to maintain and increase rates despite the economic impacts of the COVID-19 pandemic, potentially due to increased domestic tourism and reduced competition from international travel.

Netflix Data Analysis

[Tableau]

Developed a dynamic and interactive Netflix dashboard using Tableau, showcasing key insights about the platform’s content library. The dashboard provides a comprehensive view of the top 10 genres and most popular ratings, as well as a comparison between the distribution of movies and TV shows. Additionally, I created a geographic map that displays the distribution of titles by country, giving users a unique perspective on the platform’s reach. This user-friendly dashboard allows users to easily access details about thousands of titles, providing a deeper understanding of Netflix’s content offerings.
